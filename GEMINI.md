## Commands

- `/build-fix` `[error-message-or-file]`: Diagnose and fix build/type errors
- `/code-review` `[file-or-directory]`: Run a comprehensive code review on recent changes
- `/learn`: Continuous learning from feedback patterns
- `/plan` `[task-description]`: Create an implementation plan for a task
- `/quality-gate`: Run quality checks before committing
- `/reflect`: Extract learnings from the current session
- `/security-scan` `[file-or-directory]`: Quick security scan for common vulnerabilities
- `/tdd` `[feature-description]`: Start a TDD workflow (RED-GREEN-REFACTOR)
