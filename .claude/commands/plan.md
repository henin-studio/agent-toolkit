# Plan

Create a structured implementation plan for the given task.

## Steps

1. **Understand the task**: Read the task description and clarify scope.
2. **Explore codebase**: Search for existing implementations, patterns, and utilities that can be reused.
3. **Identify files**: List all files that need to be created or modified.
4. **Design approach**: Choose the best architecture considering trade-offs.
5. **Write plan**: Output a structured plan with:
   - Context section (why this change)
   - Step-by-step implementation
   - Files to create/modify
   - Verification steps
6. **Ask for approval**: Present the plan before starting implementation.
